# Rock-Paper-Scissor
# ✊✋✌️ Rock Paper Scissors Game

A fun and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. Play against the computer, test your luck, and enjoy a clean, responsive user interface!

---

## 🚀 Features

* 🎮 Play against the computer
* 🤖 Random computer choice generation
* 🧠 Instant result calculation (Win / Lose / Draw)
* 📊 Live score tracking (You vs Computer)
* 🔄 Replay anytime without refreshing
* 💡 Simple and responsive UI design

---

## 🛠️ Technologies Used

* **HTML** – Structure of the game
* **CSS** – Styling and layout
* **JavaScript** – Game logic and interactivity

## 🎯 How It Works

1. Click on one of the options:

   * ✊ Rock
   * ✋ Paper
   * ✌️ Scissors

2. The computer randomly selects its move.

3. The winner is decided based on rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock

4. Scores are updated instantly.

5. Play as many rounds as you like!

---

## 🧪 Game Logic

* Uses **event listeners** to detect user clicks
* Generates computer choice using `Math.random()`
* Compares both choices using conditional logic
* Updates DOM dynamically to show results and scores
